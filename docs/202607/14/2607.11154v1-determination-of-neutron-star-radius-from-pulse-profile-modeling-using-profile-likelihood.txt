# Determination of neutron star radius from pulse profile modeling using profile likelihood 

Vyaas Ramakrishnan 1, ∗ and Shantanu Desai 1, †

> 1

Department of Physics, Indian Institute of Technology, Hyderabad, Kandi, Telangana 502284, India 

# Abstract 

In recent years, NICER data has been extensively used to determine neutron star radius and mass using pulse profile modeling. The pulse profile modeling is implemented with the X-PSI package and best-fit parameters are obtained using Bayesian inference. Using synthetic data, we demonstrate the application of frequentist inference to determine the neutron star radius, where the nuisance parameters are treated using profile likelihood. We find that the profile likelihood technique can recover the true radius to < 1σ. Its precision is also comparable to that of Bayesian analysis while being computationally much faster. Therefore, this work serves as a proof-of-principle application of frequentist inference to determine neutron star radius using pulse profile modeling and complements the Bayesian inference technique used. We have also made our analysis codes for frequentist inference using X-PSI publicly available.

1arXiv:2607.11154v1 [astro-ph.HE] 13 Jul 2026 I. INTRODUCTION 

The Neutron Star Interior Composition Explorer (NICER) is an X-Ray satellite that has facilitated precise measurements of neutron star mass and radius through its timing preci-sion ever since its launch in 2017 [1]. The mass and radius have been determined through detailed analysis of the associated X-ray emission through pulse profile modeling for a large number of neutron stars [2, 3]. The mass and radius measurements can provide stringent constraints on neutron star equation of state [4]. The analysis of the X-ray pulse profiles is usually done using the X-ray Pulse Simulation and Inference package (X-PSI) [5]. This package is a ray tracing code, which models the phase- and energy-dependent emission from hot spots and bulk of the rotating neutron star surface. This package enables simultaneous modeling of hot-spot geometry and atmospheric emission, while incorporating both special and general relativistic effects within the oblate Schwarzschild framework. More details of the physics inputs used in X-PSI are described elsewhere [6, 7]. This package provides pa-rameter estimates for neutron star mass, radius as well as emission geometry using Bayesian inference [8]. The sampling of posteriors required for Bayesian inference is done using the 

MultiNest [9] software, based on Nested Sampling. This package has been widely used in a number of works [3, 10–12] (and references therein). However, discrepancies in the inferred radius have been found among different studies in literature using this package. For instance, Ref. [10] obtained a measurement for the equatorial radius of PSR J0740+6620 to be 12 .49 +1 .28

Similar differences in results from nested samplers have also been found in the posteriors obtained from merging compact binaries [14]. To test this conjecture, a comparison of the parameter estimation accuracy of MultiNest 

was done using another sampler, namely UltraNest [15], using synthetic data, which mimics the real NICER data of PSR J0740+6620 [13] (H25, hereafter). Ultranest is based on the Sliced Sampling Technique [16] and is known to be more robust than MultiNest , although much slower. Using numerical experiments, it was shown that both samplers produced unbiased results on parameter estimation [13]. Subsequently, it was also shown that both samplers produce consistent credible intervals on real data from PSR J0740+6620. Never-theless, some concerns about the convergence of MultiNest were raised in H25. This work 2found that convergence was challenging due to the flat likelihood surface for radii > 11 km making it difficult to constrain the upper limit. Furthermore, as the radius increases, ad-ditional degeneracies are obtained. They also cautioned that higher sampling settings were needed for real data, to fully explore the parameter space. Such tests require more compu-tational resources and hence were not carried out in H25. Furthermore, another caveat with Bayesian parameter inference is that in case a prior range for the free parameters is large, marginalization needed for parameter estimation is subject to volume effects, which could introduce a non-negligible bias into the parameter estimates [17]. In order to alleviate the aforementioned concerns with the application of Bayesian param-eter estimation in X-PSI , we demonstrate how one can use frequentist techniques based on profile likelihood to infer the neutron star radius. Although Bayesian statistics has been the industry standard in astrophysics and cosmology for the past two decades, in recent years there has been a resurgence of interest in the application of profile likelihood in parameter estimation [18–23]. We note however that particle physics and particle astrophysics still mostly use frequentist inference for parameter estimation [24]. Therefore, given the impor-tance of results from X-PSI for fundamental physics, it behooves us to compare the results obtained using Bayesian and frequentist inference, which is the goal of this work. The manuscript is structured as follows. We discuss our analysis in Sect. II, results in Sect. III, and conclude in Sect. IV.  

> II. ANALYSIS

We describe the methodology used in both the standard Bayesian and frequentist infer-ence pipelines, to compare and contrast the estimated parameter values along with their associated uncertainties. All analyses were performed using the X-PSI pulse-profile modeling framework. The Bayesian and frequentist pipelines shared an identical likelihood implementation and as-trophysical model configuration, ensuring a consistent basis for methodological comparison. We now describe each of them below. 3A. Bayesian analysis 

For Bayesian analysis, we follow the same settings as in H25. The parameter estimation was performed using the MultiNest nested sampling algorithm, interfaced within X-PSI .The adopted pulse-profile model was the ST-U configuration, consisting of two single-temperature, simply connected circular hot regions with unshared parameters. The neutron star atmosphere was modeled by the NSX fully ionized hydrogen atmosphere model [25, 26], while photon propagation was calculated using general-relativistic ray tracing in the oblate Schwarzschild spacetime approximation.  

> B. Frequentist analysis

Frequentist parameter estimation was performed using a profile likelihood framework implemented directly on top of the X-PSI likelihood evaluation module. No modifications were made to the underlying physical model. All settings are kept unchanged from [13], ensuring that differences between the Bayesian and Frequentist analyses arose solely from the inference methodology. We provide a very brief outline of parameter estimation using profile likelihood. More details and applications to astrophysics can be found in [22, 23] (and references therein). Let us consider a parameter estimation problem where the parameter vector ( θ) consists of two parameters: θ ={ϕ,α}. Among these, let us consider ϕ to be the parameter of interest and α to be the nuisance parameter. In frequentist statistics one calculates the profile likelihood, obtained by maximizing the likelihood L(ϕ, α ) with respect to α:

The optimization was carried out using the Nelder-Mead simplex algorithm as implemented in scipy.optimize.minimize .

Nevertheless, our results show that frequentist parameter estimation can complement Bayesian estimates of neutron star radius using pulse profile modeling. 7IV. CONCLUSIONS 

In this work, we present a proof-of-principles determination of neutron star radius from pulse-profile modeling with the X-PSI package, using a frequentist profile likelihood frame-work to complement the Bayesian inference technique normally used. We demonstrate using synthetic data that frequentist inference can recover the true neutron star radius to an accuracy < 1σ. Its precision is comparable to that of Bayesian analysis while being com-putationally faster than the MultiNest -based Bayesian inference used in X-PSI . In future works, we shall apply this technique to real pulsar data from the NICER telescope. In the spirit of open science, we have made our profile likelihood-based modification to X-PSI publicly available, which can be found at https://github.com/vyaas3305/ xpsi-frequentist-inference .
